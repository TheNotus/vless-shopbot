# Вклад в развитие ChronoLog

Благодарим вас за интерес к проекту ChronoLog! Мы приветствуем вклад от сообщества.

## Процесс Pull Request

1.  **Форкните** репозиторий.
2.  **Создайте ветку** для вашей функции или исправления (`feature/AmazingFeature`, `fix/BugFix`).
3.  **Сделайте коммит** ваших изменений (`git commit -m 'Add some AmazingFeature'`).
4.  **Запушьте** вашу ветку (`git push origin feature/AmazingFeature`).
5.  **Создайте Pull Request** в ветку `main` оригинального репозитория.

Пожалуйста, убедитесь, что ваш код соответствует стилю проекта и проходит все тесты.

## Кодекс поведения

Мы ожидаем, что все участники будут следовать нашему [Кодексу поведения](CODE_OF_CONDUCT.md). Пожалуйста, ознакомьтесь с ним перед тем, как вносить вклад.
